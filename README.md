Quantifying the Computing Energy Efficiency Paradox
==================================================================

### Installation

1. Install [`uv`](https://docs.astral.sh/uv/getting-started/installation/).

1. Install [Chrome](https://www.google.com/chrome/), if not already installed.

### Usage

1. Open the Jupyter Notebook with:

   ```shell
   $ uv run jupyter notebook rebound_effect_in_computation.ipynb
   ```

1. Run all cells in the Notebook.
